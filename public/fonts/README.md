# Fuentes

Coloca aquí el archivo de fuente **AvantGarde-BookTh.woff2** para que la aplicación use la tipografía correcta.

Si no tienes el archivo .woff2, puedes convertirlo desde otros formatos usando herramientas en línea como:
- https://cloudconvert.com/woff-to-woff2
- https://everythingfonts.com/

La aplicación funcionará con la fuente por defecto hasta que agregues este archivo.
